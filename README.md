## 📌 Sobre el Proyecto

Este proyecto está enfocado en la aplicación de técnicas de Business Intelligence (BI) para el análisis de ventas.

A través de herramientas como Power BI y Python, se implementaron soluciones que transforman datos crudos en un modelo relacional robusto y en Dashboards interactivos para la toma de decisiones empresariales.

## 📁 Estructura del Repositorio

Para garantizar orden y reproducibilidad, el repositorio contiene:

*   **`dashboards/`**: Archivos de Power BI (`.pbix`) que contienen las visualizaciones interactivas:
    *   `Avance_1_Limpieza_Transformacion_Lourdes_Diamela_Alarcon.pbix`
    *   `Avance_3_Dashboard_Interactivo_Lourdes_Diamela_Alarcon.pbix`
*   **`notebooks/`**: Cuaderno Jupyter con la lógica de Python utilizando Pandas para el ETL (Extracción, Transformación y Carga):
    *   `Avance_2_Modelo_Relacional_Lourdes_Diamela_Alarcon.ipynb`
*   **`data/`**: (Ignorado en Git por tamaño) Contiene los archivos crudos y transformados (`.csv` y `.xlsx`).

## 🛠️ Metodología

1.  **Limpieza y Transformación (Power BI/Power Query):** Tratamiento inicial de la base de datos de ventas, normalizando formatos y corrigiendo valores atípicos.
2.  **Modelo Relacional (Python):** Uso de Pandas en Jupyter Notebook para diseñar e implementar un modelo de datos relacional (Entity-Relationship) que optimiza las consultas analíticas.
3.  **Visualización y Storytelling:** Construcción de un Dashboard Interactivo final en Power BI que permite monitorear KPIs clave como ventas totales, evolución temporal y rendimiento por categorías.

## 🚀 Cómo ejecutarlo

1. Clona este repositorio.
2. Para explorar el modelo relacional, utiliza **Jupyter Lab** o **VS Code** para abrir el archivo en `notebooks/`.
3. Para interactuar con los Dashboards, necesitas tener instalado [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/). Abre cualquiera de los archivos ubicados en `dashboards/`.
